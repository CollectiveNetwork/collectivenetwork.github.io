#Collective Network

To get set up:
-pull the code
-get a Facebook token ID: https://developers.facebook.com/tools/accesstoken/
-npm install
-command line 'tsc' so that on save the code will build
-to auto refresh the browser, you may use this Chrome Extension: https://chrome.google.com/webstore/detail/super-auto-refresh/kkhjakkgopekjlempoplnjclgedabddk
